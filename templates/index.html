ℹ️ About This System

AI-Based Load Prediction and Optimization System

This system provides a comprehensive solution for:

  • Data Analysis: Exploratory data analysis with visualizations
  • Preprocessing: Handling missing values, outliers, and feature engineering
  • Prediction: Random Forest model for CPU load prediction

  • Pandas, NumPy (Data Processing)
